Exercice: Document de base

  1. Que manque-t-il comme information à la balise head pour que le fichier s'affiche correctement ?

    Pour moi le fichier s'affiche correctement mais en le passant par W3C Validator on nous demande d'indique le type d'encodage en ajouter un tag <meta> indiquant le type d'encodage

  2. À quoi servent les tags h1, h2 et h3 ?

    Les tags h1...h6 indiquent un titre ou sous-titre les h1 sont des titres de premier niveau à chaque fois le chiffre augmente la taille de police diminue

  3. A quoi sert le tag p?

    Le tag P est utilisé pour mettre en forme le texte dans un paragraphe.

  4. Rajouter cetter phrase à votre document

    Les tags HTML commencent par une balise ouvrante <foo> et finissent par une balise fermante </foo>.

  5. Comment en gras ou en italique?

    pour mettre en gras on utilise le tag <b>Votre texte ici!</b> gras et pour l'italique on utilise le tag <i>Votre texte ici!</i> italique

  6. Comment mettre un mot en gras et en italique au même temps?

    Pour mettre en gras et en italique à la fois on utilise les deux tags au même temps <b><i>Votre texte ici!</b></i> l'inverse marche aussi Celà est gras et italique

  7. Comment faire pour que touts les quetion soient automatiquement numérotées?

    Pour numéroter toutes les questions automatiquement il faut regrouper tous dans un tag <ol></ol> une liste ordonnée et à l'intérieur pour chaque question on le met dans un tag <li></li> qui définit un élément de liste et ce tag regroupe la question et la réponse le <h3></h3> <p></p> .

  8. Coment rajouter un titre à la fenêtre du navigateur

    on rajoute le tag <title></title> dans le head avec le titre désiré.

  9. Comment ajouter un lien vers le cours de programmation web

    Pour ajouter un lien on utilise le tag <a></a> avec l'attribut href qui indique le lien qu'on veut

    cours de programmation web

  10. Dessinez et insérez une image désigant l'arbre des différents tags correspondant à votre compte rendu

Exercice: Les listes et les tables

  1. Ecrivez la liste suivante sous forme d'une liste numérotée

    Pour cela on utilise le tag <ol></ol> une liste ordonnée

    1. maths
    2. physique
    3. français

  2. Transformez là en une liste à puces

    Pour cela on utilise le tag <ul></ul> une liste non ordonnée

    • maths
    • physique
    • français

  3. Ecrivez la table de multiplication suivante sous forme d'une table HTML avce modification

    Pour créer cette table on utilise le tag <table></table> et pour chaque ligne on utilise le tag <tr></tr> et pour les titres on utilise le tag <th></th> .

    1 2 3 4
    1 1 2 3 4
    2 2 4 6 8
    3 3 6 9 12
    4 4 8 12 16

  4. Ajouter une case faisant 2 colonnes et une case faisant deux lignes dans une nouvelle table.

    Pour faire cela on utilise l'attribut colspan pour avoir des caises plusieurs colonnes et rowspan pour avoir des lignes faisant plusieurs lignes.

    ligne 1, case 1 ligne 1, case 2
    ligne 2, case 1 ligne 2, case 2 ligne 2, case 3
    ligne 3, case 1 ligne 3, case 2 ligne 3, case 3
    ligne 4, case 1 ligne 4, case 3
