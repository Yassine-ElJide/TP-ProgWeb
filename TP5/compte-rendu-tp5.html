Exercice 1: Hello World

  1. Comment fait-on pour ouvrir la console développeur ?

    On fait la combinaison Ctrl + Shift + J sur Chrome, et sur Firefox on fait Ctrl + Maj + K ou bien on clique droit sur inspecter élément et on part sur la console.

  2. Afficher les chiffres de 0 à 99 dans la console.

    On utilise une boucle for et on affiche le chiffre à chaque itération avec le code suivant :

                    for (let i = 0; i < 100; i++) {
                    console.log(i);
                }

  3. Modifier le code pour ne pas afficher les multiples de 3.

    On ajoute la condition suivante au code pour sauter les multiples de 3 avec le mote clé continue

                    for (let i = 0; i < 100; i++) {
                    if (i % 3 === 0) continue;
                    console.log(i);
                }

Exercice 2: Damier

  1. Expliquer ce que fait la ligne "use strict";.

    Cette ligne indique au navigateur qu'il doit utilise JavaScript en "strict mode". Ceci empêche d'utiliser des variables indéfinies et par conséquent force à rédiger du beau code avec du sens.

  2. Expliquez ce que fait l'appel document.write().

    Cet appel permet d'ajouter du contenu à l'HTML de la page, directement via JavaScript et de le changer .

  3. Ajouter une fonction drawCheckerLine().

    On crée une div avec le document.write() à l'intérieur de la fonction drawCheckerLine qui prend en paramètre un entier n qui représente le nombre de cases voulu, puis avec une boucle for on appel la fonction drawCheckerCell.Code :

                    function drawCheckerLine(n) {
                    document.write('<div style="clear:both;">');
                    for (let i = 0; i < n; i++) {
                      drawCheckerCell();
                    }
                    document.write("</div>");
                }

  4. Ajouter une fonction drawCheckerBoard().

    On rajoute un boucle for à l'intérieur de la fonction qui fait appeler la fonction drawCheckerLine pour n qu'on veutCode :

                    function drawCheckerBoard(n) {
                    for (let i = 0; i < n; i++) {
                      drawCheckerLine(n);
                    }
                }

  5. Complétez le style de la balise style du document pour définir une case noire et une case blanche et utilisez ce style pour les cases du damier.

    à l'intérieur de la fonction drawCheckerLine dans la boucle for on crée une variable evenCelles pour détecter les case paires/impaires à l'aide d'un ternary Operator en faisant la somme des indices et les lignes mod 2 si c'est valid on retourne true après on passe cette valeur booléane à la fonction drawCheckerCell et on utilise un autre ternary operator pour distinguer la couleur si la valeur de la variable est true on rajoute une couleur blanche sinon noir. Code :

                    .white {
                    background-color: white !important;
                }
                  .black {
                    background-color: black !important;
                }

  6. Ajoutez un style de case rouge puis ajoutez un dernier paramètre à drawCheckerBoard() pouvant être vrai ou faux.

    Même concept pour cette question sauf que cette fois si la valeur booléane est passé à la fonction drawCheckerBoard qui elle même le passe à drawCheckerLine dont on fait l'initialisation de la variable booléane qui est passé après à drawCheckerCell. Code:

                      function drawCheckerCell(color, red) {
                    const bg = color ? "white" : "black";
                    const diagonale = red ? "red" : "";
                    document.write(`<div class="cell ${bg} ${diagonale}"></div>`);
                  }

                  function drawCheckerLine(n, ligne, diagonale) {
                    document.write('<div style="clear:both;">');
                    for (let i = 0; i < n; i++) {
                      const evenCells = (i + ligne) % 2 ? true : false;
                      const drawLine = diagonale && i === ligne;
                      drawCheckerCell(evenCells, drawLine);
                    }
                    document.write("</div>");
                  }

                  function drawCheckerBoard(n, diagonale) {
                    for (let i = 0; i < n; i++) {
                      drawCheckerLine(n, i, diagonale);
                    }
                  }
